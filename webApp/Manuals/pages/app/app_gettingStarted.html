2.1 Getting started

Login

Beim Login der Hauptapplikation meldet sich der Guide mit seinen zugeteilten Credentials an. (entsprechen nicht den Schulzugangsdaten)

Zusätzlich wird in der 3.Zeile die Server IP + Port eingegeben. --> XXX.XXX.XXX.XXX:8000

Führung starten

Bevor eine neue Führung gestartet werden kann muss die PLZ und das Geschlecht des Besuchers eingetragen werden.

PLZ einfach in das dafür vorgesehene Eingabefeld einfüllen. (Für den Fall, dass jemand keine Angabe machen will bitten wir um die Eingabe von 0000, damit die ungültigen PLZ gesammelt vorliegen)

Geschlecht wird mithilfe eines Klicks auf eines der beiden Symbole erfasst. (Links: männlich -- Rechts: weiblich)

Mit erfolgreicher Eingabe der Daten und einem Klick auf "Führung starten" kann es losgehen.

Start / Ende scannen

Die Führung startet immer beim Torbogen der Informatik Abteilung. Dort befindet sich auch der START und END Synchronisations QR-Code.

Nachdem eine Führung gestartet wurde verlangt das System nach dem START QR-Code. Ohne dem START-Scan wird kein anderer Scan (Station) angezeigt.

Mit erfolgreichem START Scan bekommt man eine Info (siehe Abbildung) und kann als nächstes schon einen richtigen Stations QR-Code scannen.


Um eine Führung zu beenden (egal ob alle Stationen besucht wurden oder nicht) muss der END QR-Code am Torbogen gescannt werden um dem System die erforderlichen Daten bzgl. der Führung zu übermitteln. Mit scannen des END Punktes erscheint wieder die Start-Ansicht.
